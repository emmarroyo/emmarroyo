Topic 7: HTML

❬❬ Back to Unit 2

What you need to know

7.1 Document structure

7.1.1 Understand how HTML is used to structure web pages:

  1. doctypes
  2. elements
  3. tags
  4. attributes

7.1.2 Understand how to declare the language of an HTML document

7.1.3 Understand how the head element is used to supply information about the document:

  1. metadata
  2. document title
  3. scripts
  4. styles
  5. links to external files

7.1.4 Be able to write organised syntax:

  1. use lowercase letters within element names, values and attributes
  2. indent nested elements
  3. use double quotes
  4. omit the values on Boolean attributes
  5. remove the forward slash at the end of self-closing elements.

7.1.5 Understand how global attributes are used to define elements:

  1. class
  2. id
  3. style
  4. accesskey
  5. lang
  6. tabindex
  7. data
  8. hidden

7.2 Structural markup

7.2.1 Understand what is meant by block-level elements and inline elements.

7.2.2 Understand that elements can belong to different content models which follow certain rules:

  1. flow
  2. sectioning
  3. heading
  4. phrasing
  5. embedded
  6. interactive

7.2.3 Be able to use some key elements to define the structure and formatting of text on a web page:

  1. articles
  2. sections
  3. headings
  4. paragraphs
  5. thematic breaks
  6. emphasis
  7. importance

7.2.4 Be able to create lists

  1. unordered
  2. ordered (and use the start and type attributes)
  3. definition/description
  4. nesting lists

7.2.5 Be able to create links:

  1. internal
  2. external
  3. email
  4. open links in a new browser window or tab
  5. link to a specific part of the same page
  6. link to a specific part of another page

7.3 Page components

7.3.1 Be able to add images to web pages:

  1. file format
  2. image size
  3. resolution
  4. retain original proportions
  5. position images
  6. alt tag.

7.3.2 Be able to represent information in a table

  1. row
  2. data
  3. headings, body and footers
  4. combine multiple cells.

7.3.3 Be able to create a form on a web page

  1. form structure
  2. form elements
  3. form controls
  4. form buttons
  5. organise and group form elements
  6. input types
  7. text areas
  8. drop-down lists.

7.3.4 Understand how web forms work

  1. how information is sent from the browser to the server
  2. form validation.

7.3.5 Be able to prepare and add audio and video

  1. multiple file formats
  2. embed in a web page
  3. controls
  4. customise controls
  5. add attributes
  6. use the source element to specify multiple resources.

7.3.6 Understand how to use inline frames to add dynamic content from external websites.
