Topic 8: CSS

❬❬ Back to Unit 2

What you need to know

8.1 Writing CSS

8.1.1 Understand the purpose of CSS:

  1. control page layout
  2. consistent page design

8.1.2 Be able to reference CSS within the HTML code:

  1. inline styles
  2. internal style sheets
  3. external style sheets
    1. folder structure
    2. naming conventions
    3. relative path

8.1.3 Understand how to write CSS rules:

  1. selectors
  2. properties
  3. values

8.1.4 Understand CSS selectors

  1. type
  2. class
  3. id
  4. universal
  5. attribute
  6. child
  7. descendant
  8. adjacent sibling
  9. general sibling
  10. multiple selectors.

8.1.5 Understand how to write efficient style sheets through general rules that apply to most elements and how to apply specific rules to individual elements:

  1. cascade
  2. inheritance

8.1.6 Understand the CSS Box Model

8.1.7 Be able to create rules using CSS attribute selectors that apply to elements that have an attribute with a specific value

  1. existence
  2. equality
  3. space
  4. prefix
  5. substring
  6. suffix

8.2 Styling web pages

8.2.1 Be able to specify colours

  1. color names
  2. hexadecimal notation
  3. RGB values

8.2.2 Understand how to manipulate colour:

  1. opacity
  2. gradients
  3. HSL values

8.2.3 Be able to specify length values:

  1. absolute lengths
  2. relative lengths.

8.2.4 Be able to add styles to elements

  1. text layout
  2. fonts
  3. links
  4. lists
  5. tables
  6. forms
  7. images

8.2.5 Be able to use the box model to add backgrounds (background images, gradients, CSS sprites) and borders to elements

8.3 Positioning content

8.3.1 Be able to control the position of elements:

  1. normal flow
  2. relative positioning
  3. absolute positioning
  4. fixed positioning
  5. floating elements
  6. overlapping elements

8.3.2 Be able to use the box model to control the appearance of boxes:

  1. display
  2. width
  3. height
  4. borders
  5. margins and padding.

8.3.3 Understand how to design for differently sized screens

  1. fixed width layouts
  2. liquid layouts
  3. layout grids
  4. CSS frameworks.

8.3.4 Understand and be able to use responsive design techniques

8.4 CSS animations

8.4.1 Be able to use CSS transitions and transforms to create animations

  1. transition properties
  2. transform properties
  3. 3D transforms
  4. cubic-bezier.
