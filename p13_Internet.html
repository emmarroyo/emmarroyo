IP addressing and DNS

➙ Craig n Dave video here

➙ TeachIct page about DNS

Points to remember:

  • Each device connected to a network is given an IP address.
  • Web pages are usually stored in remote web servers which can be identified by their IP addresses
  • When a user types into his web browser the URL of a webpage, the domain name is first sent to DNS which translates it into an IP address
  • The web browser can then send a request to the webserver (using the received IP address from the DNS)
  • The webserver resturns the requested webpage to the web browser


Static and dynamic websites

➙ Webpage explaining the difference

➙ Craig n Dave video here

➙ Slides here

Points to remember:

  • Static website are hard coded. Every user will view the same website
  • Dynamic websites are personalised based on each user. For instance based on the location of the user it will display different information. You can also create accounts, post comments etc.
  • Static websites use only client side scripting (HTML, CSS and Javascript). They can still be interactive thanks to Javascript.
  • Dynamic websites use server side scripting (PHP for instance) which can communicate with a database (storing user info) and generates the personalised HTML and CSS when the webpage is requested.

The cloud

➙ Craig n Dave video here

➙ TeachIct page about cloud storage

➙ TeachIct page about hosted applications

Points to remember:

  • Cloud storage consists in storing files in a remote server accessible via an internet connection (for example Google Drive, Dropbox etc.)
  • Cloud computing means using applications which are installed on a remote server and which can be used using an internet connection (for example Google docs, Google sheets etc)
  • Applications which are not installed locally on your computer but remotely on a server are colled hosted applications
  • Using cloud storage means that you use a cloud service provider (for isntance Google or Dropbox). You have to trust this third party company to implement suitable security measures to protect your data
