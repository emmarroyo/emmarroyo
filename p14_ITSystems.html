Points to remember:

  • A client-server network is a network which uses a central server
  • client-server networks are centralised.
  • A server is a computer configured to wait for requests from clients and execute them. For instance, a web server stores webpages. It waits for clients request to sent back the requested webpages.
  • Types of servers include: web server (stores webpages), file server (stores files centrally), email server, dhcp server (gives IP addresses to the clients)...

Points to remember:

  • Wi-Fi and 4G consume the most energy but have the largest range
  • Zigbee is very simple to implement and low power: well suited to IoT devices
  • Infrared communication needed a clear line of sight (doesn't go through obstacles or walls)
  • Ethernet is wired connection: mare stable/reliable as it is less prone to interferences and usually faster as the bandwidth is larger than wireless connections
